# Deep Learning Projects

A collection of 7 Deep Learning projects covering CNN, RNN, ANN
and Autoencoder architectures applied on real-world datasets.
All projects completed during Data Analyst Internship at
Cube AI Solutions Tech Pvt Ltd.

## Projects

### 1. Intel Image Classification using CNN
- Architecture: Convolutional Neural Network (CNN)
- Classified natural scene images into 6 categories
  (buildings, forest, glacier, mountain, sea, street)
- Tools: Python, TensorFlow, Keras, Matplotlib

### 2. E-Commerce Product Classification using CNN
- Architecture: Convolutional Neural Network (CNN)
- Classified e-commerce products into categories using images
- Tools: Python, TensorFlow, Keras, Matplotlib

### 3. Credit Card Fraud Detection
- Architecture: Deep Neural Network
- Detected fraudulent credit card transactions using DL
- Handled imbalanced dataset for better model performance
- Tools: Python, TensorFlow, Keras, Scikit-learn, Pandas

### 4. MNIST Handwritten Digit Recognition using Autoencoders
- Architecture: Autoencoder
- Reconstructed and classified handwritten digits from MNIST
- Learned compressed representations of digit images
- Tools: Python, TensorFlow, Keras, Matplotlib

### 5. Hourly Sensor Data Prediction using RNN
- Architecture: Recurrent Neural Network (RNN)
- Predicted hourly sensor readings using sequential data
- Captured time-series patterns using RNN layers
- Tools: Python, TensorFlow, Keras, Pandas, Matplotlib

### 6. Telco Customer Churn Prediction using ANN
- Architecture: Artificial Neural Network (ANN)
- Predicted telecom customer churn based on usage patterns
- Tools: Python, TensorFlow, Keras, Pandas, Scikit-learn

### 7. MNIST Image Classification using ANN
- Architecture: Artificial Neural Network (ANN)
- Classified handwritten digit images from MNIST dataset
- Compared ANN performance with standard benchmarks
- Tools: Python, TensorFlow, Keras, Matplotlib

## Common Steps in Each Project
- Loaded and preprocessed the dataset
- Built and compiled the neural network architecture
- Trained the model with appropriate epochs and batch size
- Evaluated model using accuracy and loss metrics
- Plotted training vs validation accuracy and loss curves
- Made predictions and analyzed results
